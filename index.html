Broadway Heading

A web component

The above header is an example of the <broadway-heading> web component

Example Usage

<html> <head> <script src="path/to/platform.js"> </script> <link rel="import" href="path/to/broadway-heading.html"> <style> broadway-heading h4 { margin: 0; } </style> </head> <body> <broadway-heading color="red"> <h4>Heading text</h4> <!-- h4 is used as an exmaple. Any valid html is acceptable, but the component was designed to be used with headings --> </broadway-heading> </body> </html>

Output

Heading text

Options

color
The css color-value to apply to the text and borders
